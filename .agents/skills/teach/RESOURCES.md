# AI Agent 系统 Resources

## Knowledge

- [Docs: Claude Code 官方文档](https://docs.anthropic.com/en/docs/claude-code)
  Anthropic 官方的 Claude Code 文档，涵盖 agent、hook、skill、MCP、workflow 等全部核心概念。Use for: 所有 Claude Code 概念的权威定义和配置方法。

- [Docs: Anthropic Agent SDK](https://docs.anthropic.com/en/docs/agent-sdk)
  Anthropic 的 agent 开发 SDK，是构建自定义 agent 系统的基础库。Use for: 理解 agent 的编程接口，从"用 agent"过渡到"建 agent"。

- [Article: "Building effective agents" — Anthropic Research](https://www.anthropic.com/research/building-effective-agents)
  Anthropic 官方博客，总结了构建 agent 系统的最佳实践和常见模式。Use for: 理解 agent 架构的核心理念——什么时候需要 agent，什么时候不需要。

- [Docs: Model Context Protocol (MCP)](https://modelcontextprotocol.io/)
  MCP 是一种开放协议，让 agent 安全地连接外部工具和数据源。Use for: 理解 agent 如何"接入外部世界"。

- [Repo: .claude/ 本地配置](.claude/)
  用户自己仓库中的 Claude Code 配置，包含真实的 skill 和 hook 示例。Use for: 学习材料——用自己熟悉的环境理解概念。

## Wisdom (Communities)

- [r/ClaudeCode](https://reddit.com/r/ClaudeCode)
  Claude Code 用户社区，分享配置、skill、workflow 的实践经验。Use for: 看别人怎么用 agent + skill + hook 解决实际问题。

- [Anthropic Developer Forum](https://forum.anthropic.com/)
  Anthropic 官方开发者论坛，有 SDK 和 API 讨论。Use for: 问 agent 开发相关的技术问题。

## Gaps

- "多 agent 并行"的最佳实践尚未形成共识——目前是前沿领域。社区知识分散在各处，后续需要自行收集和对比不同方案。
- 中文高质量资源较少，大部分优质文档是英文的。教学时需要平衡。
