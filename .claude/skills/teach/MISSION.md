# Mission: AI Agent 系统搭建

## Why
我想从零开始理解和搭建自己的 AI agent 系统——特别是多 agent 并行的场景。最终目标是能独立配置 agent + skill，不依赖别人替我决定架构。

## Success looks like
- 能用自己的话解释 agent、hook、skill、tool、MCP、workflow 等核心概念及其关系
- 能在 Claude Code 中独立配置和调试一个自定义 skill（含 hook）
- 能设计一个多 agent 协作的方案（哪怕只跑通一个 demo）
- 看到一个新的 AI 工具时，能识别出它用了什么 agent 模式

## Constraints
- 以 Claude Code 为学习沙盒（当前正在使用的工具，有现成的 .claude/ 配置可参考）
- 中文教学，但概念保留英文原文（方便对照文档）
- 动手优先——每节课必须有可操作的练习

## Out of scope
- 大模型训练、微调、transformer 架构内部原理
- 语言模型数学基础（概率、梯度下降等）
- LangChain、AutoGPT 等第三方框架（先学原生的 Claude Code / Anthropic SDK）
