# Git Resources

## Knowledge

- [Book: _Pro Git_ — Scott Chacon & Ben Straub](https://git-scm.com/book/en/v2)
  官方推荐的 Git 书籍，免费在线。前两章是最好的入门材料。Use for: 所有基础概念、内部原理、以及作为权威参考。

- [Interactive: _Learn Git Branching_](https://learngitbranching.js.org/)
  基于可视化的交互式 Git 教程，用图形展示 commit 和 branch 的结构。Use for: 理解分支操作、merge、rebase 的空间直觉。

- [Article: "The Git Parable" — Tom Preston-Werner](https://tom.preston-werner.com/2009/05/19/the-git-parable.html)
  用一个假想故事解释 Git 为什么被设计成这样。不是技术文档，但能帮你理解 Git 的设计哲学。Use for: 理解 Git 的"为什么"。

- [Video: "Git from the Bits Up" — Tim Berglund (YouTube)](https://www.youtube.com/watch?v=MYP56QJpDr4)
  从数据结构角度解释 Git。Use for: 视觉学习者、想深入理解 commit 图的人。

## Wisdom (Communities)

- [r/git](https://reddit.com/r/git)
  Git 专属 subreddit，中等活跃度，有经验的用户会回答疑难问题。Use for: 遇到奇怪的报错或想求证一个做法是否最佳实践。

- [Stack Overflow — git tag](https://stackoverflow.com/questions/tagged/git)
  绝大多数 Git 问题在这里都有答案。Use for: 具体报错排查。

## Gaps

- 目前缺少面向"通过 AI agent 操作 Git"的最佳实践资源——这是新兴领域，community knowledge 还在形成中。后续课程需自行设计这个视角。
