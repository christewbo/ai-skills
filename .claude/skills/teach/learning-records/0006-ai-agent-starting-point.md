# Starting point: AI agent 系统初学者

用户当前对 agent、hook、skill 只有粗略的直觉——在 Claude Code 中见过这些词，但不清楚各自的职责和关系。编程背景扎实（已在 Claude Code 中完成 Git 工作流学习），但 AI agent 架构知识为零。

目标是能独立搭建多 agent 并行系统并自行配置 agent + skill。

**Evidence:** 用户自述；在 Claude Code 中使用过 skill 和 agent 功能但不确定它们之间的关系。

**Implications:** 从最基本的概念定义开始。用 Claude Code 自身的配置（.claude/ 目录）作为第一手教学材料——用户在用的工具就是最好的教具。第一个里程碑：让用户能指着 Claude Code 中的配置说清楚"这个文件是 agent 定义，那个是 hook 配置，那个是 skill 指令"。
